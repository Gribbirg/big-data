## Текст на слайде

**Интеграция с языками программирования**

| Язык | Возможности |
|------|-------------|
| **Python** | pandas, scikit-learn, TensorFlow, PyTorch |
| **R** | ggplot2, caret, dplyr, tidyverse |
| **Java** | Собственные узлы, расширения |
| **JavaScript** | Интерактивные визуализации |

**Как это работает:**
- Python Script — выполнение Python-кода внутри workflow
- R Snippet — встраивание R-скриптов
- Двусторонний обмен данными между KNIME и скриптами

## Текст выступления

KNIME — это не закрытая система. Она отлично дружит с разными языками программирования.

Python — наверное, самая популярная интеграция. Можно использовать pandas, scikit-learn, TensorFlow, PyTorch — прямо внутри KNIME workflow.

R тоже полностью поддерживается. Весь tidyverse, ggplot2 для графиков, caret для ML — всё доступно.

На Java можно писать свои собственные узлы и расширения.

JavaScript используется для интерактивных визуализаций и веб-интерфейсов.

Как это работает? Есть специальные узлы — Python Script, R Snippet. Добавляешь их в workflow, пишешь код, и данные автоматически передаются туда-сюда между KNIME и скриптом. То есть можно комбинировать визуальный подход с кодом там, где это нужно.
