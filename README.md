## Описание

Проект содержит набор практических работ, охватывающих различные аспекты работы с большими данными:
- Основы Python и анализа данных
- Машинное обучение с scikit-learn
- Визуализация данных
- Работа с базами данных PostgreSQL
- Развертывание кластеров и мониторинг (Docker, Grafana, Prometheus)

- Python 3.11+

## Быстрый старт

### macOS/Linux

```bash
# Клонировать репозиторий
git clone <repository-url>
cd big-data

# Запустить скрипт установки
./setup.sh

# Активировать виртуальное окружение
source .venv/bin/activate

# Запустить Jupyter Notebook
jupyter notebook
```

### Windows

```cmd
# Клонировать репозиторий
git clone <repository-url>
cd big-data

# Запустить скрипт установки
setup.bat

# Активировать виртуальное окружение
.venv\Scripts\activate

# Запустить Jupyter Notebook
jupyter notebook
```

## Утилиты

### Очистка ноутбуков

Скрипт `scripts/clean_notebook.py` удаляет метаданные jetTransient, которые могут препятствовать экспорту в HTML:

```bash
python scripts/clean_notebook.py path/to/notebook.ipynb
```

Пример:
```bash
python scripts/clean_notebook.py practical_work_3/practical_work_3.ipynb
```

## Правила оформления

### Именование файлов и папок
- Использовать английский язык
- Формат snake_case для директорий и файлов
- Без кириллических символов в путях

### Jupyter Notebooks
- Без комментариев в коде
- Минимум дополнительного текста
- Структура:
  1. Номер задания
  2. Полное условие задачи
  3. Код решения
  4. Вывод результата
- Задачи с пользовательским вводом должны использовать функцию `input()`

### Структура практических работ
```
practical_work_N/
├── practical_work_N.ipynb    # Основной ноутбук
├── task/                      # Материалы задания
├── data/                      # Данные (если есть)
└── report/                    # Отчет (если требуется)
```

## Добавление новых зависимостей

При использовании новых Python библиотек необходимо обновить оба скрипта установки:
1. `setup.sh` (для macOS/Linux)
2. `setup.bat` (для Windows)

Добавьте новую библиотеку в команду `pip install`:
```bash
pip install pandas scikit-learn jupyter matplotlib seaborn numpy scipy plotly umap-learn statsmodels new-library
```
